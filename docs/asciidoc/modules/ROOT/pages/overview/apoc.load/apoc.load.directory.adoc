= apoc.load.directory
:description: This section contains reference documentation for the apoc.load.directory procedure.

label:procedure[] label:apoc-full[]

[.emphasis]
apoc.load.directory('pattern', 'urlDir', \{config}) YIELD value - Loads list of all files in folder specified by urlDir or in import folder if urlDir string is empty or not specified

== Signature

[source]
----
apoc.load.directory(pattern = * :: STRING?, urlDir =  :: STRING?, config = {} :: MAP?) :: (value :: STRING?)
----

== Input parameters
[.procedures, opts=header]
|===
| Name | Type | Description
|pattern|STRING?| The pattern describing the files that should be listed. The pattern supports '?' and '*' to represent single or multiple wildcard characters, see https://commons.apache.org/proper/commons-io/apidocs/org/apache/commons/io/filefilter/WildcardFileFilter.html[WildcardFileFilter - Apache Commons IO] for more information.
Default value: *
|urlDir|STRING?| The directory searched for files. Default value: empty String, which is replaced by the configured `import` directory
|config|MAP?|{}
|===
