= apoc.convert.toYaml
:description: This section contains reference documentation for the apoc.convert.toYaml function.

label:function[] label:apoc-extended[]

[.emphasis]
apoc.convert.toYaml(value, $config) - Serializes the given value to a YAML string

== Signature

[source]
----
apoc.convert.toYaml(value :: ANY, config = {} :: MAP) :: STRING
----

[.procedures, opts=header]
|===
| Name | Type | Default
|value|ANY|null
|config|MAP|{}
|===
