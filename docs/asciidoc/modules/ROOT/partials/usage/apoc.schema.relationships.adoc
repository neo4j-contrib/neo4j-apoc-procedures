The examples in this section are based on a database that has applied the following constraints:

[source,cypher]
----
CREATE CONSTRAINT likesDay
FOR ()-[like:LIKED]-()
REQUIRE (like.day) IS NOT NULL;
----

[source,cypher]
----
CALL apoc.schema.relationships();
----
